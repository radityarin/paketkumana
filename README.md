# paketkumana
A package tracking app using MVVM, Clean Architecture, Koin, Coroutines, Retrofit, Glide.

### Features
- Track package

### API
Cek Resi API by BinderByte
https://docs.binderbyte.com/api/cek-resi

### Library:
1. Retrofit
2. Koin
3. Coroutine
4. Glide
5. ViewBinding
